# element-theme-chalk

> element component chalk theme.

## Installation

```shell
npm i ss-ele
```

## Usage

Use Sass import

```css
@use 'ss-ele/lib/theme-chalk/index.scss';
```

Or Use vite/webpack

```javascript
import 'ss-ele/lib/theme-chalk/index.css'
```

Or

```html
<link
  rel="stylesheet"
  href="https://unpkg.com/ss-ele/lib/theme-chalk/index.css"
/>
```
